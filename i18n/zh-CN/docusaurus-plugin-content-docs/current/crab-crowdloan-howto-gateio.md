1. 打开 Gate.io 手机 App, 点击 “质押 KSM”。

   ![gate1](./assets/crowdloan/gate1.png)

2. 选择 “CRAB”，点击进入项目详情。

   ![gate2](./assets/crowdloan/gate2.png)

3. 在该页面中填写KSM的数额并点击 “立即参与” 即可。

   ![gate3](./assets/crowdloan/gate3.png)

4. 竞拍结束后，奖励将直接在 Gate 中发放，用户可在 Gate 中查看奖励。
